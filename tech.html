Tech

Some software and technologies I use.

Eclipse

As a Java developer, I have used a number of IDE's, a number of years ago I was recommended Eclipse(www.eclipse.org), many things which sold me on it was interaction with JUnit, and Hot Deploying using apache ant (ant.apache.org).

Vim

I suppose developers are divided into two groups, those who like vi, and those who like emacs, I am of the former despite the learning curve to get up and running, it always seems to be my fall back editor. Vim (www.vim.org) is the incarnation I use.
