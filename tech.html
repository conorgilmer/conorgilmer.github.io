Tech

Some software and technologies I use.

Eclipse

As a Java developer, I have used a number of IDE's, a number of years ago I was recommended Eclipse, many things which sold me on it was interaction with JUnit, and Hot Deploying using Apache Ant.

Vim

Developers are divided into two groups, those who like vi, and those who like emacs, I am of the former despite the learning curve to get up and running, it always seems to be my fall back editor. Vim is the incarnation I use.

Tor

Web browser which maintains your privacy(Tor).

GitHub

GIMP

Wordpress

Maybe against my initial thoughts, I have become a wordpress developer, and it being the market leader in CMS's it is such a useful tool for websites, especially with its plugins wordpress.org.
