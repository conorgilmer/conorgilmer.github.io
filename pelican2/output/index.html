Static site generators I have looked at recently

As an alternative to CMS's I have looked at static website generators

  • Pelican - Static site generator using python which these pages are generated by - Pelican
  • Jekyll - Static site generator using ruby - Jekyll
  • Hyde - Static site generator using python Hyde
  • Nikola - Static site generator - Nikola
  • JBake - Static site generator - JBake One thing I like about Pelican is the plugin to use LaTeX.
