IP Telephony and Mobile Phone Apps

It must be 15 years since I first heard of IP Teleophony, working on research projects for a telecom company. The it seemed to be an obvious replacement for Plain old telephone system(POTS) over the years it has become a complimentary service to POTS and other new technology which has replaced it. I was using Ekiga on Ubuntu, at a time when Skype on linux wasnt great and often needed messing with libary files to install it. I remember getting an IP/POTS hand set which could be plugged into a use port of your computer and a telephone RJ11 socket. I also recall getting buying a philips handset which was preconfigured for skype as well using wifi, I suspected it would be the way to go and you would see them everywhere. However it was mobile phones and devicesi as well as "video conferenceing" which is the major success IP teleophony, I was using skype on your mobile, tablet, laptop and desktop. I had skype on my phone and whats app, and would use IM services, but found they used so much memory - I ditched skype and whatsapp and used viber for now but expect to switch back and forth before settling on a s/w. Using google hangout and screenhero for colaborative working on projects, or using skype, gtalk, facebook chat as well as their instant messaging services has made the technology or simliar technology mainstream, almost so people dont realise they are using the technologies.
